About

This simulation tool uses the exact same calculation algorithms as the Home Assistant integration.

Features:

  • Test all three cover types: vertical blinds, horizontal awnings, and tilt/venetian blinds
  • Visualize cover position changes throughout the day
  • View solar position data (azimuth and elevation)
  • Interactive sun path diagrams
  • Export results as CSV or JSON for further analysis
