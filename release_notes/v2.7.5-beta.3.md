### Force Override Sensors
- **Weather Safety Protection**: Configure binary sensors (rain, wind, etc.) that override automatic control when active
- **Global Control**: When ANY configured sensor is active ("on"), ALL covers move to a configured safe position
- **Customizable Safe Position**:
  - Default: 0% (fully retracted/closed) - ideal for rain/wind protection
  - Configurable: Set to 100% for security sensors requiring emergency access
- **Manual Override Preserved**: Manual control still works during force override
- **Configuration Variables**:
  - `force_override_sensors`: List of binary sensor entity IDs to monitor
  - `force_override_position`: Target position (0-100%) when override is active
