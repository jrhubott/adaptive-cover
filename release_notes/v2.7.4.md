### Technical Details

**Root Cause:**
Two import statements were placed incorrectly:

1. **entity_base.py (Line 106)**: Import inside class definition
   - Broke module namespace during import
   - Caused `NameError` for `callback` decorator
   - Home Assistant module loader detected as blocking I/O

2. **button.py (Line 23)**: Import inside async function
   - Triggered blocking I/O warnings in event loop
   - Home Assistant 2026.2+ has stricter async checks (Python 3.13)

**Solution:**
- Moved `EntityCategory` import to module level in entity_base.py
- Moved `DOMAIN` import to module level in button.py
- Follows Python PEP 8 conventions for proper import organization
